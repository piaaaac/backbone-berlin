- 
  sectionname: Links
  sectiontext: |
    <aside><code>&#40;link: https://google.com text: external page link target: _blank)</code></aside>
    <br />
    
    This is an (link: https://google.com text: external page link target: _blank).
    Adding target: _blank opens the page in a new tab.
    
    <aside><code>&#40;link: single/intro text: link to a single bubble)</code></aside>
    <aside><code>&#40;link: single/projects/projectname text: link to a single bubble)</code></aside>
    <br />
    
    This is a (link: single/intro text: link to a single bubble).
    Note the page url is preceded by `single/`.
    You can use this for archived pages.
